# 00 - Vision

## One Sentence
Kosha is a resource-centric personal learning organizer that helps self-learners save, contextualize, and rediscover learning materials through meaningful topics.

## Core Philosophy
- Resources are the primary objects in Kosha. Without resources, there is no value — Topics exist only to organize them.
- A Topic is a user-defined organizational context, not a magic label that changes what a resource "is."
- Kosha organizes learning resources around meaning, not storage location (this is not a folder-based file explorer).
- Kosha is not a note-taking app, flashcard system, or general productivity tool.

## Why Kosha Exists
Modern learning is fragmented across dozens of disconnected tools. The cost isn't discovering information — it's reconnecting information later. Kosha exists to preserve the *why* behind what was saved.

## Long-term Vision
A clean, lightweight place where all technical learning lives with preserved context, so the "why" behind a saved resource is never lost.

## Success Criteria (v0.1)
- Save a resource in under 3 clicks/steps, without breaking learning flow.
- Retrieve every resource tied to a Topic (e.g. "Networking") instantly, including resources shared with other Topics.
- Recall *why* a resource was saved, from the note attached to it.
