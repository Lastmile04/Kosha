# 04 - User Stories

## Core Stories (v0.1)

**As a** self-learner
**I want to** save a new Resource (URL, title, note, type)
**So that** I preserve it with context even before assigning it to a Topic

**As a** self-learner
**I want to** create a Topic
**So that** I can group related learning materials meaningfully

**As a** self-learner
**I want to** link a Resource to one or more Topics
**So that** the same resource (e.g. Beej's Guide) can appear under both "C" and "Networking"

**As a** self-learner
**I want to** browse all Resources under a specific Topic
**So that** I can see everything I've learned about that subject in one place

**As a** self-learner
**I want to** search across all Resources (titles, URLs, notes)
**So that** I can quickly rediscover old material regardless of Topic

**As a** self-learner
**I want to** edit the note/context on a Resource
**So that** my understanding can evolve over time

**As a** self-learner
**I want to** mark a Resource's status (to_learn / in_progress / completed)
**So that** I can track my progress through saved material
